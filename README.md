# pda-install-font
pda install font

## install process

ubuntu 20.04

```
cd /usr/share/fonts/truetype/

git clone https://github.com/pichair57/pda-install-font.git

cd pda-install-font/

mv tahoma/  ../

mv THSarabunNew/  ../

cd ..

chmod -R 0755 /usr/share/fonts/truetype/tahoma

chmod -R 0755 /usr/share/fonts/truetype/THSarabunNew
```
